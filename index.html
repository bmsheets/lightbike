Welcome to Lightbike! The object of this game is to surive as long as possible by avoiding the game boundaires and light walls created by other players. Player 1 is blue and player 2 is red.

Player 1 Controls

  • Up: W
  • Left: A
  • Down: S
  • Right: D

Player 2 Controls

  • Up: Up Arrow Key
  • Left: Left Arrow Key
  • Down: Down Arrow Key
  • Right: Right Arrow Key
